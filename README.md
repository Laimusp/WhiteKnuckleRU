# White Knuckle — Русская локализация (WhiteKnuckleRU)

Неофициальный фанатский перевод игры **White Knuckle** на русский язык.
Переведён интерфейс, обучение, перки и тринкеты, записки, статистика и системные экраны.

> Перевод работает поверх **XUnity.AutoTranslator** в статическом (офлайн) режиме — в процессе игры ничего никуда не отправляется и не скачивается. Кириллица отрисовывается вшитым шрифтом.

---

## Требования

- Игра **White Knuckle** (Steam, версия для Windows).
- Больше ничего ставить заранее не нужно: загрузчик модов (BepInEx) и переводчик (XUnity.AutoTranslator) либо уже в архиве (ручная установка), либо ставятся менеджером автоматически.

---

## Установка

Есть два пути. Выбери один — **не совмещай их одновременно**.

### Путь 1. Вручную (просто распаковать в игру) — рекомендуется новичкам

1. Скачай `WhiteKnuckleRU-<версия>-manual.zip` из раздела **Releases**.
2. Открой папку игры: в Steam — правый клик по **White Knuckle → Управление → Просмотреть локальные файлы**.
3. Распакуй **содержимое** архива прямо в эту папку (с заменой, если спросит).
   Рядом с `White Knuckle.exe` должны появиться `winhttp.dll`, `oswald_ru_sdf` и папка `BepInEx`.
4. Запусти игру. Первый запуск чуть дольше обычного — это нормально (инициализируется загрузчик).

Готово — игра на русском.

**Удаление:** удали `winhttp.dll`, `oswald_ru_sdf` и папку `BepInEx` из папки игры.

### Путь 2. Через мод-менеджер (Thunderstore Mod Manager / r2modman / Gale)

1. В менеджере выбери игру **White Knuckle**.
2. Найди мод **WhiteKnuckleRU** (автор Laimusp) и нажми **Install** — зависимости (BepInExPack) поставятся сами.
3. Запускай игру кнопкой **Start modded** из менеджера.

> Ставишь до публикации в каталоге? Используй **Import local mod** и укажи `WhiteKnuckleRU-<версия>-thunderstore.zip`.

## Что переведено

- Главное меню, настройки, экраны смерти/результатов, статистика забега.
- Обучение (подсказки, туториалы).
- Перки и тринкеты — включая описания с числами (перевод «на источнике», без англо-русской каши).
- Записки и лор, найденные на уровнях.
- Системные экраны игрового «ПК» (рабочий стол, программы, бэкапы).

## Чего перевод НЕ трогает (и это нормально)

- Внутренние имена объектов, коды, отладочные строки и dev-консоль (открывается только вручную) — игрок их в обычной игре не видит.
- Имена разработчиков и сообщества — оставлены латиницей намеренно.

Нашёл непереведённый или кривой текст на обычном экране? Сделай скриншот и заведи **Issue** на GitHub — поправим.
