# Русификатор для White Knuckle

Привет! Это неофициальный фанатский перевод игры White Knuckle на русский. Делал я его сам вместе с ИИ (нейросетью).

**Важно:** дальше я его поддерживать и обновлять не буду. Кто хочет — спокойно форкай [репозиторий на GitHub](https://github.com/Laimusp/WhiteKnuckleRU) и продолжай, перевод свободный, не жалко. Игра ещё в раннем доступе и обновляется, так что со временем что-то может поехать — это нормально. Нашёл английский кусок или ошибку — пиши в Issues, но чинить не обещаю (см. выше).

## Что переведено
- Меню
- Обучение
- Перки и тринкеты
- Записки
- Статистика
- Экраны игрового «компьютера»

В игре ничего не качается из интернета — перевод просто лежит в файлах.

## Установка

> ⚠️ **Важно:** не используй оба способа сразу — выбери один.

### Способ 1 — вручную (самый простой, для большинства)
1. Скачай архив **WhiteKnuckleRU-1.0.0-manual.zip** из раздела **Releases**.
2. Открой папку игры: в Steam правый клик по **White Knuckle → «Управление» → «Просмотреть локальные файлы»**.
3. Распакуй **содержимое** архива прямо в эту папку и согласись на замену. Рядом с `White Knuckle.exe` должны появиться файл `winhttp.dll`, файл `oswald_ru_sdf` и папка `BepInEx`.
4. Запусти игру. Первый запуск чуть дольше обычного — это нормально.

**Удаление:** удали из папки игры `winhttp.dll`, `oswald_ru_sdf` и папку `BepInEx`.

### Способ 2 — через мод-менеджер (r2modman / Gale / Thunderstore Mod Manager)
1. В менеджере выбери игру **White Knuckle**.
2. Нажми **«Import local mod»** и укажи архив **WhiteKnuckleRU-1.0.0-thunderstore.zip**. Всё нужное менеджер подтянет сам.
3. Запускай игру кнопкой запуска из менеджера.

*Когда мод появится в каталоге Thunderstore — можно будет просто найти его по названию и нажать Install.*
